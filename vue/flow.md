## 从0搭建项目
1. 版本管理控制
    - git
    - svn
2. node
    - nvm
    - npm
    - yarn
    - pnpm
3. 代码编辑器
    - sublime
    - notepad++
    - visual studio code
    - webstorm
4. 开始开发
    - npm init vue@lastest
    - 使用官方脚手架搭建项目
    - 配置配置文件
        - eslint
        - gitignore
        - prettierrc
        - postcssrc
        - editorconfig
    - npm install
    - npm run dev
    - 代码编写完成
    - npm run build
    - CI/CD(持续集成/持续部署)
